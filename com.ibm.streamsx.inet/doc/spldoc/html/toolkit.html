General Information

com.ibm.streamsx.inet.rest

This namespace includes a number of operators that embed the Jetty webserver to provide HTTP REST style api to streaming data. A main use is the HTTPTupleView operator that provides HTTP GET access to the contents of a stream, as JSON data. This then allows browser-based visualization of streaming data using JavaScript widgets, such as Dojo Charting.

Also provided is the ability to inject tuples into a stream using HTTP POST with the HTTPTupleInjection and HTTPXMLInjection operators.

Meta-data for the streams connected to the operators is also available through HTTP GET requests, to allow JavaScript applications to self-discover the format of the data.

Jetty 8.1.3 from Eclipse 4.2.2 is embedded in the toolkit, under opt/eclipse4.2.2.
